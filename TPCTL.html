Implementation of a CTL model checker.

This page is meant as a course/tutorial on symbolic CTL model-checking.

Apply the following steps :

  1. Download and compile libddd, libits and the ctl packages with this script and this script
  2. Edit the top of the script to have $SDD point somewhere "reasonable" on your account.
  3. Execute the scripts
  4. Replace the file : ctl/src/mc/ctlCheck.cpp with this one this version: edit everywhere marked with TODO. The algorithms to implement are given in these slides
  5. Run the control to make sure your solution is correct : in folder ctl/tests/ run the script run_all.sh. Or run "run_test.pl test_XX.data". Any "test failed" message indicates a problem. Use this set of test data, simply rm *.data and use the files in this archive instead.

NB : all tests are configured to be run with option "--backward" since you are not asked to implement the forward operators.

Notations of libddd, where A, B and C are SDD (State in the CTL tool)

  • C = A + B : union of A and B
  • C = A - B : set difference of A and B
  • C = A * B : intersection of A and B
  • State::null : the empty set
  • getReachable() : a function returning the set of all reachable states.

T,U and V are transition relations (Transition::):

  • getNextRel() : returns the transition relation of the system.
  • getPredRel() : returns predescessor transition relation of the system (backward fire).
  • C = T(A) : apply T to A, returns successors by a step of T
  • Transition::id : the identity homomorphism
  • V = T + U : a transition that returns the sum, for all A, V(A) = T(A) + U(A)
  • V = T * U : a transition that returns the intersection, for all A, V(A) = T(A) * U(A)
  • V = T & U : a transition that returns the composition, for all A, V(A) = T( U(A) )
  • U = fixpoint(T) : build a homomorphism that applies T to fixpoint.
